# 操作方法

## 基本操作

| コントローラー | キーボード＆マウス | 動作 |
| --- | --- | --- |
| START | Q | ポーズ |
| 左スティック<br>十字キー | ＷＡＳＤ | 移動 |
| L3 | Ctrl | ミニマップ切替 |
| A | 左クリック | 攻撃 |
| B | スペースキー | 憑依する |
| Y | C | 人間状態に瞬時に戻る
（人間状態以外） |

## UI操作

| コントローラー | キーボード＆マウス | 動作 |
| --- | --- | --- |
| 左スティック<br>十字キー | ＷＡＳＤ | メニュー移動 |
| A | Z | 決定 |
